# Bot Discord de Ticket
> Bot com diversas funções de ticket

Este bot para discord foi feito utilizando a biblioteca da Discord.Js juntamente com Node.Js para criar tickets com diversos comandos.

## Instalando

Para você instalar o bot você tera de ter o Node.Js instalado em sua maquina para rodar o mesmo, assim você ira dar os seguintes comando no console.

```shell
npm install path
```
```shell
npm install fs
```
```shell
npm install discord.js
```

Coloque seu token no arquivo config.json e de o seguinte comando.

```shell
node .
```

Após isso ja estará funcionando!
